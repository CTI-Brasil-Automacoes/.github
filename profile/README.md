# 🛡️ CTI Brasil — EY

Bem-vindo ao repositório central da equipe de **Cyber Threat Intelligence (CTI)** da **EY Brasil**.

Este espaço reúne os scripts, automações e ferramentas desenvolvidas internamente pelo time para apoiar as operações de CTI no dia a dia.

---

## 📦 Repositórios

- **[Analista_CTI_BOT_Telegram](https://github.com/CTI-Brasil-Automacoes/Analista_CTI_BOT_Telegram)** — Bot de Cyber Threat Intelligence que monitora feeds de segurança, filtra por relevância e envia análises automáticas para um grupo no Telegram.

- **[Script_Coleta_De_Dados_API_SHODAN](https://github.com/CTI-Brasil-Automacoes/Script_Coleta_De_Dados_API_SHODAN)** — Script para consulta à API do Shodan, permitindo a coleta e exportação de dados de exposição de ativos a partir de queries personalizadas.

- **[Script_Coleta_de_indicadores_de_Ransomware](https://github.com/CTI-Brasil-Automacoes/Script_Coleta_de_indicadores_de_Ransomware)** — Script que consome a API do Ransomware.live para extrair indicadores de ataques e atividades de grupos de ransomware.

- **[Script_Priorizacao_E_Tratativas_CVEs](https://github.com/CTI-Brasil-Automacoes/Script_Priorizacao_E_Tratativas_CVEs)** — Script para enriquecimento e priorização de CVEs, consumindo as APIs públicas do NIST NVD, EPSS (FIRST.org) e CISA KEV para consolidar score CVSS, probabilidade de exploração e status de exploração ativa em um único relatório.

- **[Script_Analise_De_Dominios_Typosquatting](https://github.com/CTI-Brasil-Automacoes/Script_Analise_De_Dominios_Typosquatting)** — Script para análise de domínios suspeitos e detecção de typosquatting, consolidando dados de WHOIS e reputação do VirusTotal em um único relatório para suporte a investigações de Threat Intelligence.

---

## ⚠️ Aviso

Todos os repositórios desta organização são de uso **interno e restrito** à equipe CTI Brasil — EY. O compartilhamento de qualquer script, chave de API ou dado sensível com pessoas externas ao projeto deve ser previamente autorizado pelo responsável da equipe.
